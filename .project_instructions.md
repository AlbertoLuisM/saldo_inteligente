# Saldo Inteligente - Project Instructions\n\n## Overview\nPersonal financial management web app in Brazilian Portuguese (pt-BR).\nTagline: \"Sua vida financeira, clara e sob controle.\"\n\n## Tech Stack\n- Next.js 14 + TypeScript\n- Tailwind CSS (emerald green theme, dark navy sidebar)\n- PostgreSQL + Prisma ORM\n- NextAuth.js (email/password)\n- Recharts for data visualization\n- Inter font family\n\n## Visual Identity\n- Sidebar: `#0f172a` (dark navy)\n- Accent/Primary: `#10b981` (emerald green)\n- Content area: `#f8fafc` (light gray)\n- Danger: `#ef4444` (red)\n- Warning: `#f59e0b` (amber)\n- Info: `#3b82f6` (blue)\n- All text in Brazilian Portuguese\n- Currency: R$ format (pt-BR Intl.NumberFormat)\n\n## Route Structure\n- `/login`, `/signup`, `/esqueci-senha` - Auth pages\n- `/(app)/dashboard` - Main dashboard\n- `/(app)/receitas` - Income management\n- `/(app)/despesas` - Expense management\n- `/(app)/dividas` - Debt management\n- `/(app)/contas-pagar` - Accounts payable\n- `/(app)/contas-receber` - Accounts receivable\n- `/(app)/metas` - Financial goals\n- `/(app)/saude-financeira` - Financial health score\n- `/(app)/relatorios` - Reports & charts\n- `/(app)/consultor-ia` - AI advisor (rule-based)\n- `/(app)/notificacoes` - Notifications\n- `/(app)/configuracoes` - Settings/profile\n- `/(app)/seguranca` - Security\n- `/(app)/visao-financeira` - Financial overview\n\n## API Routes\n- `/api/dashboard` - Dashboard KPI data\n- `/api/incomes`, `/api/incomes/[id]`\n- `/api/expenses`, `/api/expenses/[id]`\n- `/api/debts`, `/api/debts/[id]`\n- `/api/accounts-payable`, `/api/accounts-payable/[id]`\n- `/api/accounts-receivable`, `/api/accounts-receivable/[id]`\n- `/api/goals`, `/api/goals/[id]`\n- `/api/notifications`, `/api/notifications/[id]`\n- `/api/user`, `/api/user/password`\n- `/api/signup`, `/api/auth/[...nextauth]`\n\n## Demo Credentials\n- Email: demo@saldointeligente.com.br\n- Password: Demo@123\n\n## Dark Mode
- Implemented via `next-themes` with `class` strategy (Tailwind `darkMode: ['class']`)
- Toggle button (Sun/Moon icons) in the header component
- Default theme: light; user preference persists in localStorage
- CSS variables for dark mode defined in `globals.css` under `.dark` selector
- Dark background: `#0c1222`, cards: `rgba(30,41,59,0.6)`, borders: `rgba(51,65,85,0.5)`
- Sidebar stays dark navy in both themes (no changes needed)
- Progress bar tracks use `.progress-track` utility class for theme-aware styling
- All pages updated with `dark:` Tailwind classes for backgrounds, text, borders, inputs, hovers

## Key Components\n- `components/layout/sidebar.tsx` - Collapsible dark sidebar\n- `components/layout/header.tsx` - Top header with notifications\n- `components/ui/modal.tsx` - Reusable modal\n- `components/ui/form-field.tsx` - Form field wrapper\n- `components/ui/status-badge.tsx` - Status badges\n- `components/dashboard/bar-chart.tsx` - Recharts bar chart\n- `components/dashboard/donut-chart.tsx` - Recharts donut chart\n- `components/dashboard/line-chart.tsx` - Recharts line chart\n\n## Seeded Data\nRealistic 12 months of Brazilian financial data for demo user:\n- Salary R$ 8,500/mo + freelance income\n- Housing, food, transport, health expenses\n- 3 active debts (car financing, credit card, personal loan)\n- 4 financial goals\n- 6 accounts payable, 3 accounts receivable\n- 5 notifications\n
